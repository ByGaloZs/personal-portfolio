# Personal Portfolio

Personal portfolio built with Next.js, TypeScript and Tailwind CSS, focused on software engineering, robotics and industrial automation.

This project is designed as a modern, scalable portfolio using a bento-style layout to showcase selected projects, case studies and professional experience.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion (planned)
